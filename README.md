# 검사기 라이브 뷰어

OBS 3840×1080 가로 4분할 유튜브 라이브를 **한 링크**로 보여주는 페이지입니다.

- **여주 / 양산 / 광주** — 해당 공장 화면만 꽉 차게
- **전체** — 4분할 전체 화면

## 설정

`config.js`에서 유튜브 라이브 ID만 바꾸면 됩니다.

```js
youtubeVideoId: "실제_영상_ID",
```

## GitHub Pages 링크

배포 후 주소:

`https://<GitHub아이디>.github.io/obs-live-viewer/`

이 링크 하나를 사용자에게 공유하면 됩니다.
